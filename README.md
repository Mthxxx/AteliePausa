# 🏺 Pausa Ateliê

Um site moderno e responsivo para um ateliê artesanal, desenvolvido com foco em experiência do usuário, design elegante e navegação intuitiva.

## ✨ Sobre o projeto

O **Pausa Ateliê** é uma aplicação web que apresenta produtos artesanais de forma visual e organizada, valorizando o conceito de tempo, cuidado e autenticidade.

O projeto foi desenvolvido com atenção aos detalhes, buscando transmitir a essência do trabalho manual através da interface.

---

## 🚀 Funcionalidades

- 🖼️ Catálogo de produtos
- ⭐ Seção de destaques na página inicial
- 🔍 Visualização de detalhes dos produtos via modal (sem redirecionamento de página)
- 📱 Layout 100% responsivo (mobile, tablet e desktop)
- 💬 Integração com WhatsApp para contato direto
- 🎨 Interface moderna com Tailwind CSS

---

## 🛠️ Tecnologias utilizadas

- React.js
- TypeScript
- Tailwind CSS
- React Router DOM
