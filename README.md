---

## ✨ Sobre o projeto

O **Pausa Ateliê** é uma aplicação web que apresenta produtos artesanais de forma visual e organizada, valorizando o conceito de tempo, cuidado e autenticidade.

A interface foi pensada para transmitir a essência do trabalho manual, com um design limpo, cores suaves e uma experiência fluida.

---

## 🚀 Funcionalidades

* 🖼️ Catálogo de produtos
* ⭐ Destaques na página inicial
* 🔍 Modal de visualização de produtos (sem troca de página)
* 📱 Layout totalmente responsivo
* 💬 Integração com WhatsApp
* 🎨 Design moderno com Tailwind CSS

---

## 🧠 Destaque técnico

Um dos principais diferenciais do projeto é a implementação de um **modal reutilizável para produtos**, garantindo:

* Melhor experiência do usuário
* Navegação fluida sem reload
* Código limpo e reutilizável
* Componentização eficiente em React
